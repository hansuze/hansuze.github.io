通过模糊来弱化背景

使用滤镜来实现，但是如果把滤镜效果应用到body元素上，页面中的所有元素都会被模糊处理，我们想要的凸显出来的那个关键元素也不例外。所以我们需要一个额外的HTML元素来实现这个效果，把页面上除了关键元素外的一切都包裹起来，这样就可以只对这个容器的元素进行模糊处理了。

尝试一下吧！
另外：使用滤镜方案产生阴影效果，意味着一旦不支持，我们将没有任何回退方案，因此阴影效果不妨用其他方案来实现。例如上面提到的 box-shadow方案

范围输入控件方案

Css resize的方案表现很好，而且只需要极少的代码，不过，他还是有一些不足之处：
1.对键盘来说是不可访问的。
2.调整上层的唯一方法是拖动，但是对于较大的图片，这样很累，不支持点击某个点就可以调整完成
3.用户只能在上层图片的右下角操作，用户很容易忽略它。
