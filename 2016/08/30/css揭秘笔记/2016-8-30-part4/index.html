单侧投影

多数人使用box-shadow的方法是，指定三个长度值和一个颜色值
box-shadow: 2px 2px 4px rgba(0, 0, 0, .5); 接下来的几个步骤很好滴以图形化的方式讲解了投影是如何绘制的。
1.以该元素相同的尺寸和位置，画一个rgba(0, 0, 0, .5)的矩形。
2.把他向右偏移 2px 向下偏移2px
3.使用模糊算法将它进行4px的模糊处理
4.接下来，模糊后的矩形与原始元素的交集部分会被切掉，因此看起来像是在元素的后面（这跟元素在投影的上层略有不同）
现在我们介绍 box-shadow的第四个长度参数 。它排在 模糊半径之后，成为扩张半径，这个参数会根据你指定的值去扩大或者缩小投影的尺寸。 举例来说 一个 -5px 的扩张半径会把投影的高度和宽度各减少 10px;

不规则投影

难题

当我们想给一个矩形或者其他能用border-radius 生成的形状加投影时，box-shadow的表现都看成完美，但是当元素添加了一些伪元素或者半透明的装饰之后，他就有些力不从心了，border-radius会无耻的忽视这部分。这类情况包括：

  • 半透明图像，背景图像 或者border-image
  • 元素设置了点状，虚线或者半透明的边框，但没有背景（或者当background-clip 不是border-box 时）
  • 对话气泡，它的小尾巴通常是伪元素生成的
  • 我们在“切角效果”一节见过的切角形状
  • 几乎所有的折角效果，包括“折角效果” 一节提到的例子；
  • 通过clip-path形成的形状，比如”菱形图片”那一节提到的菱形图像

解决方案

滤镜效果规范为这个问题提供了一个解决方案，它引入了一个叫做filter的属性，这个属性是从SVG那里借鉴过来的。我们只需要一些函数就可以很方便的指定滤镜效果了，比如blur()、 grayscale() 以及我们需要的drop-shadow(),如果你喜欢，甚至可以把多个滤镜串联起来，只要用空格把他们分隔开就可以了。
filter: blur() grayscale() drop-shadow();
drop-shadow 滤镜的参数基本上跟box-shadow 属性是一样的，但不包括 扩张半径，不包括inset关键字(向内)

css滤镜的最大好处在于，他们可以平稳退化，当浏览器不支持时，不会出现问题，只不过是没有效果而已，如果你需要回退支持，你可以同时附上一个SVG滤镜

另外需要牢记的事情就是，任何非透明的部分都会被一视同仁的打上投影，包括文本（如果背景是透明的），那我们是不是可以通过 text-shadow:none;来取消文本上的投影呢？ 其实text-shadow跟他是完全不相干的。
此外，如果你使用text-shadow后又使用了drop-shadow() 滤镜就会为投影再打上投影。

染色效果

基于滤镜的方案

我们使用的第一个滤镜是sepia() 他会给图片一种降饱和度的橙黄色染色效果,几乎所有的像素的色相值都会被收敛到35-40

如果这是我们想要的 就可以收工了，不过我们通常需要将主色调的饱和度比这更高，可以用saturate()给每个像素提升饱和度。

图片看起来不错，但我们并不希望把图片调为这种橙黄色调，这时我们需要再添加一个hue-rotate滤镜，把每个像素的色相以指定的度数进行偏移。如果这个效果需要:hover来切换，我们甚至可以添加动画效果。

基于混合模式的方案

滤镜方案是行之有效的，但你可以能会注意到他产生的结果与我们在图像处理软件中得到的效果不完全一致。如果尝试在saturate()滤镜中增加饱和度，又会得到一种不自然 风化的效果。不过幸好，我们有混合模式。
当两个元素叠加时：混合模式控制了上层元素的颜色和下层颜色进行混合的方式。
luminosity混合模式会保留尚岑那个元素的HSL亮度信息，并从他的下层吸取色相和饱和度信息。
如果我们在下层准备好我们想要的主色调，并把待处理的图片放在上层并设置为这种混合模式，那本质上不就是在做染色处理吗？
方案1 把图片包裹在一个容器中，并把容器的背景色 设为我们想要的主色调

方案2 不用图片元素 而是用div元素，把这个元素的第一层设置要染色的图片，并把第二层设置为我们的主色调(遗憾的是我并没有测试成功…)

毛玻璃效果

尝试使用透明度

解决方案：在伪元素上使用blur实现
假设大背景的background-attachment 的值是fixed 这种情况是有可能的，对一个伪元素进行blur处理，然后将其定位到元素下层，它的背景将会无缝匹配背景。

1.为了方便调试，我们将伪元素加入了颜色,我们发现伪元素在main的上面，我们使用z-index: -1 移到main后面

2.我们使用了blur(20px)滤镜，这跟我们的效果非常接近了。模糊效果中间的部分非常完美，但是到边缘区域后会逐渐消退。这是因为模糊效果会削减实色像素所覆盖的范围，削减的幅度正是模糊半径的长度，对伪元素加上一个red的背景色帮助我们了解真相。

3.最后我们使用负margin来修复边缘模糊消退的问题，但现在的情况是有一圈模糊效果超出了容器。不过幸运的是这个问题很容易解决 只要在main上应用overflow:hidden;

最后我们总结一下步骤：
1.使用伪元素 绝对定位 并且置z-index: -1;
2.在伪元素上使用blur(20px) 滤镜
3.为了补偿使用blur边界模糊的情况 使用-margin:30px(大于20px)把模糊边界挤出去
4.使用overflow:hidden 将模糊的部分剪切掉。

折角效果

45度折角解决方案

1.首先使用渐变实现翻折效果

2.增加一个暗色的三角形来实现翻折效果。实现的方法是增加另一层渐变来生成这个三角形。

3.由于渐变的尺寸是沿着渐变轴进行度量的，所以第一层底色的渐变要除以根号2 也就是 2 /根号2 = 1.41em

其他角度的解决方案

1.现实生活中的折角往往不是精确的45°。，例如我们需要一个150°的折角，我们不仅需要重新角度，还需要定义高度和宽度

2.上面的三角形已经跟切角对上号了，但这个效果看起来更不真实了，原因是折边是需要旋转的，但是我们使用渐变是达不到这种效果的。这时候我们需要使用伪元素。

3.我们需要向上偏移 宽度-高度 的距离 并旋转30度
